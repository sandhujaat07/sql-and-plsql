Table Creation

To create a table in SQL, we use the CREATE TABLE statement. Syntax: 

CREATE TABLE table_name (
  column1 datatype [optional_constraint],
  column2 datatype [optional_constraint],
  ...
  columnN datatype [optional_constraint]
);
Example: 

CREATE TABLE employees (
  id INT PRIMARY KEY,
  name VARCHAR(50),
  age INT,
  salary DECIMAL(10, 2)
);
Output: 

Table created.

Inserting Values

To insert values into a table, we use the INSERT INTO statement. Syntax: 

INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);
Example: 

INSERT INTO employees (id, name, age, salary)
VALUES (1, 'John Doe', 30, 50000.00);
Output: 

1 row inserted.

Updating

To update existing records in a table, we use the UPDATE statement. Syntax: 

UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
Example: 

UPDATE employees
SET salary = 55000.00
WHERE id = 1;
Output: 

1 row updated.

Deleting and Dropping

To delete records from a table, we use the DELETE FROM statement. Syntax: 

DELETE FROM table_name
WHERE condition;
Example: 

DELETE FROM employees
WHERE id = 1;
Output: 

1 row deleted.

To drop a table from the database, we use the DROP TABLE statement. Syntax: 

DROP TABLE table_name;
Example: 

DROP TABLE employees;
Output: 

Table employees dropped.

Querying Database Table

To retrieve data from a database table, we use the SELECT statement. Syntax: 

SELECT column1, column2, ...
FROM table_name
WHERE condition;
Example: 

SELECT * FROM employees;
Output: 

id  |  name       |  age  |  salary
----|-------------|-------|---------
1   |  John Doe   |  30   |  50000.00
2   |  Jane Smith |  28   |  60000.00
3   |  Mark Johnson | 32  |  55000.00

Aggregate Functions and Grouping

Aggregate functions perform a calculation on a set of values and return a single value. Examples of aggregate functions include COUNT, SUM, AVG, MIN, and MAX. Syntax: 

SELECT aggregate_function(column_name)
FROM table_name
WHERE condition;
Example: 

SELECT COUNT(id) AS total_employees FROM employees;
Output: 

total_employees
---------------
3

We can also use GROUP BY clause to group rows that have the same values into summary rows. Syntax: 

SELECT column1, aggregate_function(column2)
FROM table_name
WHERE condition
GROUP BY column1;

Creation and Deletion of Views

A view is a virtual table based on the result set of an SQL statement. It contains rows and columns, just like a real table. Syntax to create a view: 

CREATE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;
Example: 

CREATE VIEW employee_details AS
SELECT id, name, salary
FROM employees
WHERE salary > 50000.00;
Output: 

View created successfully.

To delete a view, we use the DROP VIEW statement. Syntax: 

DROP VIEW view_name;
Example: 

DROP VIEW employee_details;
Output: 

View employee_details dropped.

Grant and Revoke Commands

Grant command is used to give privileges to users or roles. Syntax: 

GRANT privilege_name
ON object_name
TO {user_name | PUBLIC | role_name};
Example: 

GRANT SELECT, INSERT, UPDATE, DELETE
ON employees
TO john_doe;
Output: 

Privileges granted.

Revoke command is used to take back privileges from users or roles. Syntax: 

REVOKE privilege_name
ON object_name
FROM {user_name | PUBLIC | role_name};
Example: 

REVOKE SELECT, INSERT, UPDATE, DELETE
ON employees
FROM john_doe;
Output: 

Privileges revoked.

Commit and Rollback

Commit command is used to save the changes made by the current transaction. Syntax: 

COMMIT;
Example: 

          -- Initiating a transaction
          BEGIN TRANSACTION;
          
          -- Performing SQL statements
          
          -- Committing the transaction
          COMMIT;
          
          -- Alternatively, rolling back the transaction
          ROLLBACK;
Output: 

          Transaction committed successfully.
          Transaction rolled back.

Rollback command is used to undo the changes made by the current transaction. Syntax: 

ROLLBACK;
Example: 

ROLLBACK;
Output: 

Changes rolled back.

Joins

Joins are used to combine rows from two or more tables, based on a related column between them. Examples of joins include INNER JOIN, LEFT JOIN, RIGHT JOIN, and FULL JOIN. Syntax: 

SELECT column1, column2, ...
FROM table1
INNER JOIN table2
ON table1.column_name = table2.column_name;
Example: 

          SELECT employees.id, employees.name, departments.department_name
          FROM employees
          INNER JOIN departments
          ON employees.department_id = departments.id;
Output: 

id	name	     department_name

 1	John Doe	      HR
 2	Jane Smith     	IT
 3	Mark Johnson	  HR
 4	Alice Lee	IT

SET Operators

SET operators are used to combine the result sets of two or more SELECT statements. Examples of SET operators include UNION, UNION ALL, INTERSECT, and MINUS. Syntax: 

SELECT column1, column2, ...
FROM table1
UNION
SELECT column1, column2, ...
FROM table2;
Example: 

SELECT employee_id, employee_name
FROM employees
UNION
SELECT manager_id, manager_name
FROM managers;
Output: 

      employee_id 	employee_name

        1	            John Doe
        2	            Jane Smith
        3	            Mark Johnson
        4	            Alice Lee
        101	          Michael Scott
        102	          Miranda Priestly
        103	          Gordon Gekko
        104            	Don Draper
